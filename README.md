# BetterNS2
Mod to allow extensive customization of interfaces for Natural Selection 2  

**Modid: 8fb78f73**  
Only /src/ is pushed to the mod  
To activate locally, create a shortcut to the NS2 executable and add `-game "PATH_TO_BetterNS2/src"`
